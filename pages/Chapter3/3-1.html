3.1 Understanding Decimals

Decimals

Decimalsare numbers that represent parts of a whole. Unlike whole numbers, decimals include values that are less than one. Our decimal system is based on powers of ten, dividing whole numbers into tenths, hundredths, thousandths, and so on.

Decimal Point:The decimal point (.) separates the whole number part from the fractional part.

Place Values:Digits to the right of the decimal point represent fractions with denominators that are powers of ten.

Decimal Place Values

Just like whole numbers have place values (ones, tens, hundreds), decimals have place values too.


Example

In 5.973

- 5 is in the ones place.
- 7 is in the tenths place.
- 9 is in the hundredths place.
- 3 is in the thousandths place.

Reading and Writing Decimals

Reading Decimals

1. Whole Number Part: Read the whole number part as usual.
2. "And": Use the word "and" to represent the decimal point.
3. Decimal Part: Read the decimal part as a whole number.
4. Place Value: State the place value of the last digit.

Example

124.123: One hundred twenty-four and one hundred twenty-three thousandths.

12.003214: Twelve and three thousand two hundred fourteen millionths.

0.3263: Three thousand two hundred sixty three ten thousandths.

4.5:Four and five tenths.

0.00001One hundred thousandth.

Writing Decimals

1. Whole Number: Write the whole number part.
2. Decimal Point: Write the decimal point (.).
3. Decimal Part: Write the decimal part, ensuring the last digit is in the correct place value.

Example

Seven and ninety-two hundredths: 7.92

One and sixty-three hundredths: 1.63

Three and twenty-five hundredths: 3.25

Four and seventy-one thousandths: 4.071

One hundred thousand seven and fourteen millionths:100007.0000

Ordering Decimal

Sometimes, you'll need to compare and order decimals. Here are two helpful techniques:


1. Extra Zeros:

- Add zeros to the end of the decimals so they all have the same number of decimal places.
- This makes it easier to compare the numbers vertically.

2. Compare Place Values:

- Start by comparing the tenths place.
- Group numbers with the same tenths digit.
- Continue comparing the next place value (hundredths, thousandths, etc.).
