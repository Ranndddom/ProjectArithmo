4.3 Multiplication and Division of Integers

Multiplication of Integers

Rules

Positive x Positive = Positive
Negative x Negative = Positive
Positive x Negative = Negative
Negative x Positive = Negative

Here's a simple way to remember it:

Same signs, positive result.
Different signs, negative result.

Let's break it down step-by-step:

Ignore the signs: Multiply the numbers as if they were both positive. Determine the sign: Use the rules above to decide if your answer should be positive.

Signs matter:Pay close attention to the signs of the numbers you're working with. Same sign, positive result: When multiplying or dividing two numbers with the same sign (both positive or both negative), the result is always positive. Different signs, negative result: When multiplying or dividing two numbers with different signs (one positive and one negative), the result is always negative. Let me know if you'd like more examples or have any other questions.

Examples of multiplying integers

Here are some examples of multiplying integers, covering all the sign combinations:


1. Positive x Positive


Example: 7 x 3 = 21

Both numbers are positive, so the result is positive.


2. Negative x Negative


Example: (-5) x (-4) = 20

Both numbers are negative, so the result is positive.


3. Positive x Negative


Example: 9 x (-2) = -18

One number is positive, the other is negative, so the result is negative.


4. Negative x Positive.


Example: (-6) x 8 = -48

One number is negative, the other is positive, so the result is negative.


Division Of Integres

Rules

Positive ÷ Positive = Positive
Negative ÷ Negative = Positivec
Positive ÷ Negative = Negative
Negative ÷ Positive = Negative

The same sign rules apply:

Same signs, positive result.
Different signs, negative result

How to divide integers


Ignore the signs:Divide the numbers as if they were both positive.

Determine the sign:Use the rules above to decide if your answer should be positive or negative.

Important Notes

Dividing by zero is undefined: You cannot divide any number by zero. It's a mathematical no-no! Integers are not always closed under division: This means that when you divide two integers, theresult is not always another integer. For example, 7 ÷ 2 = 3.5, and 3.5 is not an integers

Example of division of integers

1. Positive ÷ Positive


Example: 20 ÷ 4 = 5

Both numbers are positive, so the result is positive.


2. Negative ÷ Negative


Example: (-18) ÷ (-3) = 6

Both numbers are negative, so the result is positive.


3. Positive ÷ Negative


Example: 30 ÷ (-5) = -6

One number is positive, the other is negative, so the result is negative.


4. Negative ÷ Positive


Example: (-24) ÷ 8 = -3

One number is negative, the other is positive, so the result is negative.
